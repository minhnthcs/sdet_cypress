# sdet_cypress

1. Install cypress on your machine:
- npm init -y
- npm i yarn -g (install yarn on your machine globally)
- yarn add -D cypress  (install cypress)

2. Open cypress
- open from node_modules/cypress
- npx cypress open
- modify package.json 
"script" : {
    "test":"cypress open"
}
Then run command: npm run test 
OR: yarn test
